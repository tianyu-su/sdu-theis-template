# sdu-theis-template
山东大学硕（博）士学位论文-WORD模板
本WORD模板按照学校撰写指南，设置了论文全部格式，直接使用即可。

## 模板参照
《山东大学研究生学位论文撰写指南.pdf》，发布时间：2022年1月5日，查阅日期：2023年1月20日

## 文件说明
1. 《学位论文模板-教程.pdf》给出了模板所提供方法的说明书。
2. 《学位论文模板-教程.docx》给出了PDF的原版文件，可以在写作过程中查阅如何使用或者先行学习。

## 使用方法
1. 运行 `导入样式.bat`。
2. 使用Word软件打开《山东大学硕士研究生学位论文模板.dotx》，开始自己的文章撰写。
3. 定稿后可以运行 `还原样式.bat`，也可不运行。
